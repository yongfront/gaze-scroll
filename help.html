🚨 카메라 권한 문제 해결

❌ "카메라 권한이 거부되었습니다" 오류

카메라 권한이 거부되었을 때 다음과 같은 방법으로 해결할 수 있습니다:

🔧 확장 프로그램 오류 해결

Permission 'camera' is unknown 오류:

  1. chrome://extensions/ 접속
  2. "손 제스처 스크롤 컨트롤러" 우클릭 → 제거
  3. 확장 프로그램 다시 로드 또는 재설치

Content Security Policy 오류:

  1. chrome://extensions/에서 확장 프로그램 새로고침
  2. 브라우저를 완전히 종료하고 다시 시작
  3. 시크릿 모드에서 시도 (Ctrl+Shift+N)
  4. 개발자 도구에서 캐시 비우기 (F12 → Network → Disable cache)

카메라 접근 실패 DOMException:

  1. 카메라가 다른 프로그램에서 사용중인지 확인 (Zoom, Teams 등 종료)
  2. 브라우저 권한 재설정 (주소창 왼쪽 🔒 아이콘)
  3. 시스템 설정에서 카메라 권한 확인
  4. 카메라 드라이버 업데이트

🔧 브라우저별 해결 방법

🌐 Chrome/Chromium 브라우저:

  1. 주소창 왼쪽 🔒 아이콘 클릭
  2. 카메라 권한을 '허용'으로 변경
  3. 페이지 새로고침 (Ctrl+R 또는 F5)

🦊 Firefox 브라우저:

  1. 주소창 왼쪽 🔒 아이콘 클릭
  2. 연결 보안 → 자세히 보기
  3. 권한 탭에서 카메라 '허용'으로 변경

🍎 Safari 브라우저:

  1. Safari 환경설정 → 웹사이트 → 카메라
  2. 현재 웹사이트의 카메라 권한 '허용'으로 변경

💡 추가 해결 방법

  • 시크릿 모드에서 시도: Ctrl+Shift+N (또는 Cmd+Shift+N)
  • 확장 프로그램 재설치: chrome://extensions/ 에서 제거 후 재설치
  • 브라우저 캐시 정리: chrome://settings/clearBrowserData
  • 다른 카메라 앱 종료: Zoom, Teams 등 다른 프로그램에서 카메라 사용 확인

🔧 문제 해결

손이 인식되지 않아요

  • 카메라에 손이 잘 보이도록 가까이 대세요
  • 조명이 충분한 곳에서 사용하세요
  • 손가락을 명확하게 펴거나 접으세요

스크롤이 너무 느려요/빨라요

  • 설정에서 스크롤 속도를 조절하세요
  • 민감도를 조절하여 제스처 인식 빈도를 변경하세요

확장 프로그램이 작동하지 않아요

  • chrome://extensions/ 에서 확장 프로그램이 활성화되어 있는지 확인
  • 페이지 새로고침을 해보세요
  • 확장 프로그램을 다시 로드해보세요
