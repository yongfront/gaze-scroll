# 손 제스처 스크롤 컨트롤러 🎯🖐️

**브라우저 전체에서 작동하는 혁신적인 손 제스처 스크롤 시스템!**

## ✨ 새로운 기능 (브라우저 확장프로그램)

- **🌐 모든 탭 지원**: 현재 보고 있는 어떤 웹사이트에서도 작동!
- **🎭 고급 손 제스처**: 10가지 이상의 다양한 손 모양 인식
- **⚡ 빠른 스크롤**: 각 제스처마다 최적화된 스크롤 속도
- **📱 플로팅 모드**: 웹페이지 위에 떠 있는 컴팩트한 인터페이스
- **🎯 정밀 제어**: 한 손가락부터 세 손가락까지 세밀한 제어

## 🎮 지원하는 손 제스처들

### 👆 한 손가락 제스처

- **엄지 ✊** → 맨 위로 이동
- **검지 👆** → 빠른 아래 (600px)
- **중지 🖕** → 위로 (400px)
- **약지 🤞** → 아래로 (500px)
- **새끼 👌** → 빠른 위 (800px)

### ✌️ 두 손가락 제스처

- **평화 ✌️** → 빠른 아래 (800px)
- **평화 터치 🤏** → 아래 (400px)
- **락 🤘** → 빠른 위 (1000px)

### 🤟 세 손가락 이상 제스처

- **세 손가락 🤟** → 매우 빠른 위 (1500px)
- **네 손가락** → 맨 아래로 이동

### ✊ 손 접기 제스처

- **주먹 ✊** → 위로 스크롤 (500px)

### 🖐️ 손바닥 제스처

- **손바닥 펴기** → 위치에 따른 위/아래 스크롤

## 🚀 브라우저 확장프로그램 설치하기

### Chrome 브라우저에서 설치

1. **개발자 모드 활성화**

   - Chrome 주소창에 `chrome://extensions/` 입력
   - 우측 상단의 "개발자 모드" 토글 켜기

2. **확장프로그램 로드**

   - "압축해제된 확장프로그램 로드" 버튼 클릭
   - 이 프로젝트 폴더 선택 (`/Users/yong/gaze-scroll`)
   - 📁 폴더 선택 시 프로젝트 루트 폴더를 선택하세요

3. **설치 완료 확인**

   - 확장프로그램 목록에서 "손 제스처 스크롤 컨트롤러" 확인
   - 브라우저 툴바에 🎯 손 아이콘이 나타남
   - 아이콘이 보이지 않으면 퍼즐 조각 아이콘을 클릭해서 툴바에 추가

4. **카메라 권한 사전 설정 (권장)**
   - 확장프로그램 아이콘을 우클릭 → "확장프로그램 관리"
   - "사이트 액세스" → "카메라" → "허용"으로 변경

### 🎯 설치 후 즉시 사용법

1. **아이콘 클릭** 🎯
2. **"▶️ 제어 시작" 버튼 클릭**
3. **카메라 권한 허용** (거부 시 **"🔄 카메라 권한 재요청"** 버튼 사용)
4. **실시간 카메라 화면 확인** (팝업에 표시됨)
5. **손 제스처로 스크롤 시작!**

### ⚠️ 문제 해결

**아이콘이 안 보이나요?**

- 퍼즐 조각 아이콘(확장프로그램) 클릭
- "손 제스처 스크롤 컨트롤러"를 찾아서 툴바에 고정 📌

**카메라 권한 오류?**

- 주소창 왼쪽 🔒 아이콘 클릭
- "사이트 설정" → 카메라 → "허용"으로 변경

**스크롤이 안 되나요?**

- 손가락을 3개 이상 펴세요
- 카메라가 손을 잘 비추고 있는지 확인
- 팝업에서 "제어 시작"이 활성화되어 있는지 확인

**카메라 권한 오류가 계속되나요?**

- **🔄 카메라 권한 재요청 버튼**을 클릭해보세요
- 확장프로그램을 제거하고 재설치해보세요
- 브라우저를 완전히 재시작해보세요
- 다른 브라우저(Chrome, Edge)에서 시도해보세요

**카메라 권한 재요청 방법:**

1. 팝업에서 **"🔄 카메라 권한 재요청"** 버튼이 나타남
2. 버튼을 클릭하면 카메라 권한을 다시 요청
3. 브라우저 팝업에서 **"허용"**을 선택
4. 성공하면 바로 카메라가 작동하기 시작!

- **🔄 실시간 탭 전환**: 다른 탭으로 이동해도 계속 작동
- **⚡ 즉각 반응**: 제스처 인식 후 0.1초 내 스크롤
- **🎨 시각 피드백**: 각 제스처마다 다른 색상 인디케이터
- **🔄 카메라 권한 재요청**: 권한 거부 시 다시 요청 가능
- **📱 실시간 카메라 표시**: 팝업에서 카메라 화면 확인
- **🎯 스마트 제스처 인식**: 피부톤 기반 실제 이미지 분석

## ⚙️ 설정 옵션

- **민감도**: 시선 각도 감지 민감도 (5° ~ 45°)
- **스크롤 속도**: 자동 스크롤 속도 (20px ~ 100px)
- **카메라 뷰**: 시선 추적 화면 표시 여부
- **자동 스크롤**: 시선 기반 자동 스크롤 기능

## 🛠️ 기술 스택 (브라우저 확장프로그램)

- **확장프로그램**: Chrome Extension Manifest V3
- **Frontend**: React 18 + HTML/CSS/JavaScript
- **UI**: Custom CSS + 이모지 아이콘
- **손 제스처 인식**: MediaPipe Hands API
- **카메라**: WebRTC getUserMedia API
- **통신**: Chrome Extension Message Passing API
